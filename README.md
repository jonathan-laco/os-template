# os-template 🛠️

O **os-template** é uma automação em **HTML, CSS e JavaScript** para auxiliar o time de suporte N1 na criação de ordem de serviço.

✨ O projeto foi desenvolvido para apoiar atendimentos simples e complexos com informações relevantes, trazendo mais clareza no dia a dia da equipe.

✅ A proposta é oferecer um recurso funcional para o time de suporte, que precisava de auxílio nessa etapa por falta de padronização no processo.
